Welcome to the Calendar!

Welcome to the Calendar feature of our CRM system! Adding events is easy - just follow these steps:

Adding an Event:
Double-click on the desired date where you want to add the event.
Event Details:
A pop-up will appear. Enter the event's details, such as the title, description and time.
Saving the Event:
Click "Save" to add the event to the calendar.

That's it! You've successfully added an event using the double-click function on the calendar. If you have any questions, feel free to ask.
